# Vite Vue3 ts tsx elementplus
优点本地开发速度快
### 注意事项

```
建议vscode开发需要安装eslint插件 保存自动格式化 tsx支持暂时有点问题 
```

### 安装

```bash
$ npm install -g create-vite-app
$ npm install
# or
$ yarn add -g create-vite-app
$ yarn install
```

### 运行

```bash
$ npm run dev
$ npm run build
# or
$ yarn dev
$ yarn build
```

 参考：https://github.com/cangshudada/vite-vue3-tsx
